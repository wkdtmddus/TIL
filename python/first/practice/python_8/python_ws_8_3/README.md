### animal 클래스 상속 cat 클래스 Lv3
목표  
- 객체 지향 프로그래밍에 대해 이해한다.
- 파이썬 클래스의 기본적인 개념에 대해 이해한다.
- 예외 상황에 대해 이해하고 대처한다.

문제  
ws_8_1에서 작성한 animal클래스를 상속받는 cat클래스를 작성하시오.  

요구사항  
cat클래스는 animal클래스의 속성과 메서드를 상속받는다.  
cat클래스에 추가로 meow메서드를 작성하여 "야옹!"이라는 메세지를 출력하시오.
```
야옹!
```
