### 합집합 set, union() Lv1
목표  
- 파이썬의 데이터 구조(세트, 딕셔너리)에 대해 이해한다.
- 파이썬 모듈의 정의와 활용 그리고 함수에 대해 이해한다.
- 파이썬 클래스 메서드에 대해 이해한다.

문제  
주어진 셋에서 두 개의 셋을 합친 결과를 반환하는 union_sets함수를 작성하시오.  
두 개의 셋을 인자로 받아 합집합을 반환해야 한다.   

요구사항  
없음
```
{1, 2, 3, 4, 5}
```
