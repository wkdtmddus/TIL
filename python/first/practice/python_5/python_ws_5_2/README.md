- 작성한 코드를 실행하고 발생하는 오류를 해결하고 문제를 해결한다.

문제

리스트를 인자로 받아 중복이 제거된 새로운 리스트를 반환해야 한다.

요구사항  
없음
```
[1, 2, 3, 4, 5]
```
---
여러가지 방법으로 코드를 만들 수 있습니다.  
`set`을 사용하여 중복을 제거하거나, `count()`를 사용하여 리스트안에 요소의 갯수를 세어서 확인해 추가하거나 생략할 수 있습니다.
