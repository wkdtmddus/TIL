### Vue3와 DRF 연동하기 - 인증 시스템 1 Lv2
목표  
- 회원가입 및 로그인 기능을 Vue 프로젝트와 DRF 서버에서 구현한다.
- 회원가입 및 로그인 시 토큰을 반환하고, 이를 로컬 스토리지에 저장한다.
---
문제  
DRF 서버로부터 게시글 정보를 불러 올 수 있는 Vue 프로젝트를 작성중에 있다.  
인증서비스를 포함한 프로젝트를 만들고자 한다.  
주어진 코드를 수정하여 회원가입, 로그인 기능을 구현하시오.  

요구사항
1. DRF 프로젝트에서 회원가입, 로그인 기능을 구현한다.
  - 회원가입, 로그인 시 인증에 사용 할 수 있는 token을 반환하여야 한다.
2. Vue 프로젝트에서 회원가입, 로그인 기능을 구현한다.
  - SignUpView와 SignInView 컴포넌트를 작성한다.
  - 모든 페이지에서 회원가입, 로그인 경로로 이동 가능한 링크가 표시되어야 한다.
  - 회원가입, 로그인 기능을 구현한다.
    - 회원가입, 로그인이 완료되면 반환받은 토큰을 local Storage에 저장한다
    - 회원가입, 로그인이 완료되면 HomeView 컴포넌트로 경로 이동 되어야 한다.
3. 로그인이 되어있는 경우, 회원가입, 로그인 컴포넌트로 이동시 HomeView로 이동되어야 한다.
  - beforeEnter를 활용하여 구현한다.
---
