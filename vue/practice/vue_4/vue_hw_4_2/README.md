### Vite 활용하기 1 Lv2
목표  
- Vite를 활용하여 Vue.js 프로젝트를 생성하는 방법을 이해한다.
- 생성된 프로젝트를 실행하여 환경이 올바르게 설정되었는지 확인한다.
---
문제  
Vite를 활용하여 프로젝트를 생성하고자 한다.  
제시된 이미지를 참고하여 동일한 환경의 프로젝트를 생성하고, 실행 결과를 확인하시오.  

요구사항
- 없음
---
`Vite`를 사용하여 프로젝트를 생성합니다.
- vite 프로젝트 생성
```
$ npm create vue@lastest
```
- 프로젝트 폴더 이동
```
$ cd vue-project
```
- 패키지 설치
```
$ npm install
```
- Vue 프로젝트 서버 실행
```
$ npm run dev
```
